◇ CAPABILITIES

Everything a graph database browser
should be — with the motion to match.

MOTION

Force-directed canvas

A handcrafted physics simulation. Drag nodes, watch the graph settle, zoom from a single node to thousands.

UX

Live node search

Find any node instantly by label. The canvas locks on and highlights the match in real time.

ALGORITHM

Shortest path

BFS-based path tracing — watch the teal corridor light up between any two entities.

QUERY

Cypher-lite playground

A familiar pattern syntax. MATCH (a:Person)-[r:FOUNDED]->(b:Company) — the matches glow on the canvas.

EXTRACT

Entity extraction

Paste any text — an article, a paragraph, a paper. The engine pulls entities and relationships and plots them node by node.

INSPECT

Node inspector

Click any node to see its properties, labels, and full relationship list in the side panel.
